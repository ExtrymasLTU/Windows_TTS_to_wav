# Windows_TTS_to_wav

Uses Windows Text to Speach to create .wav file of what you told it to say. It uses your already installed TTS voices.

Create TTS audio files on your Windows PC without the need of any website or 3rd party program.

Open the program in PowerShell, type in what you want it to say, set file name, choose voice, volume and speed and you get .wav file of what you wanted.
Only 2 lines are required. What you want it to say and file name after that you can press Enter to skip selections.
Voice selection default is the 1st voice in the list.
Volume default is 100 and speech rate default is 0.
Have fun :D
